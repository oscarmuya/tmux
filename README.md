## Clone the tmux repo
```
git clone https://github.com/oscarmuya/tmux ~/.config/tmux
```

## Remove the .git folder
```
rm -rf ~/.config/tmux/.git
```

## Start tmux
```
tmux
```

## Install plugins via TPM
```
<C-Space> I
```


## Install pomodoro
```
~/.config/tmux/install-pomodoro.sh
```
